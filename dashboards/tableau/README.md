# Tableau Dashboard

Open `zepto_inventory_dashboard.twb` in Tableau Public or Tableau Desktop. It is configured as a Tableau workbook starter for the cleaned dashboard CSV exports.

Recommended source table: `data/dashboard/fact_products.csv`.

The dashboard preview images are stored at `outputs/tableau_dashboard_preview.svg` and `outputs/tableau_dashboard_preview.png`. The `.twb` file is the Tableau workbook, while the preview image is the picture shown in the repository README.
